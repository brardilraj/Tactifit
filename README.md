# ⚽ Tactifit — AI Football Player Scouting & Recommendation System

Tactifit is a data-driven scouting platform that recommends football players based on club requirements such as budget, position, and age.  
The system combines web scraping, data processing, machine learning, and a REST API to assist clubs in making intelligent recruitment decisions.

---

## 🚀 Key Features

✅ Player recommendation based on similarity to club needs  
✅ Budget, age, and position filtering  
✅ Automated data collection from football statistics websites  
✅ Machine learning–based clustering & dimensionality reduction  
✅ REST API for retrieving player recommendations  
✅ Data-driven scouting insights  

---

## 🧠 Machine Learning Techniques Used

- PCA (Principal Component Analysis)
- K-Means Clustering
- Similarity Matching (Euclidean Distance)
